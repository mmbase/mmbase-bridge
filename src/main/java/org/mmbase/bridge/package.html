The main interface to MMBase, also known as `MMCI' (MMBase Cloud Interface). One of the core interfaces of this package is indeed {@link org.mmbase.bridge.Cloud}, which gives direct access to MMBase data.

Associated with the bridge are also the searchquery interfaces, and the security interfaces

Script languages can access MMBase in this way. JSP can e.g. do this rather directly, or a little less directly by use of the MMBase Taglib.

@since MMBase-1.4 @see org.mmbase.storage.search.SearchQuery
